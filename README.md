# Cabinet Dentaire Moderne

Site web générique et moderne pour un cabinet dentaire professionnel.

## Fonctionnalités

- **Accueil** : Section Hero avec phrase d'accroche et boutons d'action
- **À propos** : Description générique du dentiste et points forts
- **Services** : Liste complète des services dentaires typiques
- **Galerie** : Images génériques de cabinet dentaire moderne
- **Témoignages** : 3 avis fictifs de patients satisfaits
- **Localisation & Contact** : Adresse générique, téléphone, horaires et formulaire
- **Footer** : Informations génériques du cabinet

## Personnalisation

Tous les contenus sont facilement modifiables :
- Remplacer `[Nom]` par le nom du dentiste
- Mettre à jour l'adresse et les coordonnées
- Changer les images selon le cabinet
- Adapter les horaires d'ouverture

## Technologies

- React avec TypeScript
- Tailwind CSS
- Vite
- Composants UI modernes

## Installation

```bash
npm install
npm run dev
```

Le site est prêt à être personnalisé pour n'importe quel dentiste.
